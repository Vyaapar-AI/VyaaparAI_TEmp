# **App Name**: Sweet Delights Bakery

## Core Features:

- Product Catalog: Browse available baked goods with images, descriptions, and prices.
- Shopping Cart: Add, remove, and manage items in a virtual shopping cart.
- Secure Checkout: Complete purchases with a secure payment gateway.
- Order History: View past orders and track their status.
- AI-Powered Recommendation Tool: AI algorithm to analyze user behavior and suggest relevant products based on browsing history and preferences.

## Style Guidelines:

- Primary color: Pastel Pink (#FFD1DC) for a soft, inviting feel. It evokes the delicate nature of baked goods and confections.
- Background color: Cream (#FAF0E6), a very desaturated hue near the primary color. Provides a warm and comforting backdrop that enhances readability and visual appeal.
- Accent color: Light Brown (#D2B48C). Use it on buttons, highlights, and accents, contrasting nicely with the primary and background colors.
- Headline font: 'Playfair', a serif with a fashionable, high-end feel; Body font: 'PT Sans', a humanist sans-serif. These fonts work well together.
- Use simple, line-based icons in brown or pastel pink to represent product categories and UI elements.
- Use a clean, grid-based layout with ample spacing to showcase products effectively.
- Subtle hover effects and transitions to provide a delightful user experience.
